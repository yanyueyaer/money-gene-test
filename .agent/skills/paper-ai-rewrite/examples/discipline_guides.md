# 学科差异改写指南

> 不同学科有不同的写作传统和规范，改写时必须尊重学科特性。本指南按学科类型提供针对性建议。

---

## 一、理工科（计算机、电子、机械、材料等）

### 写作特点
- 语言简洁直接，少用修辞
- 数据驱动，重定量分析
- 公式和符号密集
- 图表使用频繁

### 改写策略

| 可改写部分 | 改写方式 |
|-----------|---------|
| 引言/背景 | 增加具体技术演进细节，用数据替代笼统描述 |
| 方法描述 | 增加实验设置的具体参数和选择理由 |
| 结果分析 | 增加对数据趋势的个性化解读和比较 |
| 讨论/结论 | 增加与相关工作的具体对比分析 |

### 不可改写部分
- ⛔ 算法伪代码和流程
- ⛔ 数学公式及其推导
- ⛔ 实验数据本身
- ⛔ 标准技术术语（如"卷积层"、"梯度下降"）

### 示例
```
❌ 本文提出了一种基于深度学习的方法，实验结果表明该方法效果显著。
✅ 本文构建了一个三层 BiLSTM 架构，隐藏层维度设为 256。在 CIFAR-10 上测试时，
   准确率从基线的 91.2% 提升至 94.8%（p < 0.01）。这一提升主要归因于双向
   结构对上下文信息的更充分利用。
```

## 二、人文社科（文学、历史、哲学、社会学等）

### 写作特点
- 论证丰富，逻辑链条较长
- 引用其他学者观点频繁
- 允许较多主观分析和评论
- 语言风格相对灵活

### 改写策略

| 可改写部分 | 改写方式 |
|-----------|---------|
| 理论框架 | 用对话式语气讨论不同学者观点，增加批判性评论 |
| 文献回顾 | 按思想脉络组织而非时间顺序，增加学术评价 |
| 分析论证 | 使用更多实例、类比和反证，增加论证层次 |
| 结论 | 增加反思性讨论和展望 |

### 可以大胆使用的手法
- ✅ 个人学术立场表达（"笔者以为"、"依笔者拙见"）
- ✅ 批判性评论（"此说虽有道理，但忽略了…"）
- ✅ 引文点评（"正如 XXX 精辟地指出…"）
- ✅ 反问和设问

### 示例
```
❌ 关于这一问题，学界存在不同的观点。一些学者认为…另一些学者则提出…
✅ 围绕这个问题，学界的态度可以说是泾渭分明。以 Smith（2019）为代表的一派
   坚持认为…而 Johnson（2020）则唱了反调，他的核心论据在于…有意思的是，
   双方都援引了同一组数据，却得出了几乎相反的结论——这本身就很说明问题。
```

## 三、经济管理类

### 写作特点
- 定量与定性分析并重
- 案例分析使用频繁
- 模型构建和假设检验是核心
- 语言较为规范化

### 改写策略

| 可改写部分 | 改写方式 |
|-----------|---------|
| 研究背景 | 引入具体市场数据、政策事件或企业案例 |
| 理论基础 | 增加理论之间的比较和适用条件讨论 |
| 模型分析 | 增加变量选择的理由说明和替代方案讨论 |
| 政策建议 | 增加可操作性细节和实施条件分析 |

### 不可改写部分
- ⛔ 统计检验结果的数值
- ⛔ 模型方程和变量定义
- ⛔ 财务数据和指标名称

### 示例
```
❌ 研究结果对企业管理实践具有重要的启示。
✅ 对于年营收在 5000 万以下的中小企业而言，这一发现意味着：与其盲目扩大广告
   投入，不如将资源集中在老客户的精细化运营上——我们的数据显示，客户留存率
   每提升 5%，利润增幅可达 25%-95%（参考 Reichheld, 2001 的经典估算）。
```

## 四、医学/生物科学

### 写作特点
- 术语高度专业化
- 表述必须精确严谨
- 遵循 IMRAD 结构（引言-方法-结果-讨论）
- 伦理声明和知情同意等不可省略

### 改写策略

| 可改写部分 | 改写方式 |
|-----------|---------|
| 引言 | 增加疾病流行病学数据和临床意义说明 |
| 方法 | 增加操作细节和质控措施描述 |
| 结果 | 增加对数据的临床意义解读 |
| 讨论 | 增加与其他研究的比较和临床转化讨论 |

### 绝对不可改写部分
- ⛔ 疾病名称、药物名称（尤其是通用名）
- ⛔ 剂量、给药方案
- ⛔ 检验指标名称和参考范围
- ⛔ 诊断标准引用
- ⛔ 伦理审批编号和声明内容
- ⛔ 统计学结果数值

### 示例
```
❌ 本研究探讨了某药物对某疾病的治疗效果，结果表明具有显著疗效。
✅ 治疗组接受瑞舒伐他汀 10mg/d 口服，疗程 12 周。结果显示，治疗组的
   LDL-C 水平从基线的 4.2±0.8 mmol/L 下降至 2.1±0.5 mmol/L（降幅
   50.0%），与安慰剂组差异具有统计学意义（t=5.32, p<0.001）。从临床角度看，
   这一降幅已达到最新版指南推荐的目标值。
```

## 五、法学

### 写作特点
- 逻辑论证严密
- 法条引用精确
- 案例分析是重要论证手段
- 语言较为正式和规范

### 改写策略

| 可改写部分 | 改写方式 |
|-----------|---------|
| 问题提出 | 用具体案例引出法律问题 |
| 法理分析 | 增加比较法视角，对比不同法域的做法 |
| 制度评析 | 增加批判性分析和完善建议的具体操作方案 |

### 不可改写部分
- ⛔ 法律条文原文引用
- ⛔ 判例要旨
- ⛔ 法律术语（如"善意取得"、"不当得利"）

---

## 六、教育学

### 写作特点
- 理论与实践结合紧密
- 常用调查研究和行动研究
- 受众意识强，表述较为通俗
- 案例和经验总结使用频繁

### 改写策略
- 增加具体的教学案例或场景描述
- 加入一线教师或学生的视角和声音
- 用数据支撑而非笼统评价
- 增加实践操作层面的细节

---

## 通用原则

1. **术语一致性** — 全文中同一概念使用同一术语，不要为了多样性而混用
2. **学科规范优先** — 当"降低 AI 率"与"学科规范"冲突时，以学科规范为准
3. **引用格式** — 严格遵守目标期刊或学位论文要求的引用格式
4. **查阅范文** — 改写前建议先阅读目标学科顶刊的范文，感受学科写作风格
