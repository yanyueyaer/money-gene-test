# 改写模式与示例库

> 本文档提供常见的改写模式和具体操作手法，供改写过程中参考。

---

## 模式一：句式翻转

### 原理
改变句子的主干结构，打破 AI 生成文本的句式规律性。

### 操作手法

#### 1.1 主动 ↔ 被动转换
```
原文：研究人员使用了深度学习方法来分析数据。
改写：数据的分析工作借助了深度学习方法（主语变化）。
改写：深度学习方法被引入到数据分析环节当中（被动化）。
```

#### 1.2 语序调整
```
原文：为了提高模型的准确率，本文引入了注意力机制。
改写：注意力机制的引入，是本文提升模型准确率的关键手段。
改写：本文引入注意力机制，其出发点在于提高模型准确率。
```

#### 1.3 从句位置变换
```
原文：在人工智能快速发展的背景下，自然语言处理技术取得了突破性进展。
改写：自然语言处理技术近年来取得了突破性进展，这与人工智能整体的快速发展密不可分。
```

## 模式二：长短句交替

### 原理
AI 生成文本的句子长度通常在一个较窄的范围内波动。通过制造长短句的明显交替，增加文本的"爆发度"。

### 操作手法

#### 2.1 拆长为短
```
原文：本研究采用了问卷调查与深度访谈相结合的研究方法，对来自不同地区的300名受访者进行了数据收集，并运用统计分析软件对数据进行了系统处理。（67字）

改写：
本研究综合运用了问卷与深度访谈两种方式。受访者共300人，来源覆盖多个地区。数据收集完成后，借助统计软件进行了系统化处理。（拆为3句：20字+15字+20字）
```

#### 2.2 合短为长
```
原文：
实验结果表明模型精度有所提升。提升幅度约为5%。这一结果具有统计显著性。

改写：
实验结果表明，模型精度提升了约5%且该提升幅度经统计检验证实具有显著性。（合为1句）
```

#### 2.3 节奏制造
```
原文（均匀）：
该方法具有较高的计算效率。同时它还能有效降低噪声干扰。此外该方法的可扩展性也较好。

改写（有节奏）：
该方法计算效率高。这一点在大规模数据集上尤为突出——不仅噪声干扰被有效压低，其可扩展性也经实验证实表现优异。一句话概括：快、稳、可扩展。
```

## 模式三：具体化替代泛化

### 原理
AI 倾向于使用泛化、概括性的表述，人类写作则更常使用具体的例子、数据和细节。

### 操作手法

#### 3.1 添加具体数据
```
原文：近年来，深度学习在图像识别领域取得了显著进展。
改写：自 2012 年 AlexNet 在 ImageNet 竞赛中将错误率从 26% 降至 16% 以来，深度学习在图像识别领域的进展可谓日新月异。
```

#### 3.2 替换笼统引用
```
原文：大量研究表明该方法是有效的。
改写：Zhang 等人（2021）和 Li 等人（2022）的对照实验均证实了该方法在提升分类准确率方面的有效性。
```

#### 3.3 增加具体场景
```
原文：该技术可以应用于多个领域。
改写：该技术已在医学影像辅助诊断（如肺结节检测）、自动驾驶（车道线识别）等场景中落地应用。
```

## 模式四：人味注入

### 原理
加入人类写作中常见但 AI 不太会生成的元素。

### 操作手法

#### 4.1 评论性插入
```
原文：实验结果表明该方法的准确率达到了 95.3%。
改写：实验结果颇为理想——准确率达到 95.3%，这一数字甚至略超出了我们的预期。
```

#### 4.2 转折与反差
```
原文：尽管该方法效果良好，但计算复杂度较高。
改写：该方法的效果确实让人眼前一亮，不过先别急着下结论——它的计算复杂度是个不小的问题，尤其是在资源受限的场景下。
```

#### 4.3 学术口语化
```
原文：这一现象可以从以下几个方面进行解释。
改写：要解释这一现象，不妨从几个角度来拆解。
```

#### 4.4 设问引导
```
原文：本节将对实验结果进行分析。
改写：实验数据已经摆在面前，关键问题是：这些数字背后到底说明了什么？
```

## 模式五：结构重组

### 原理
改变段落内部和段落之间的组织方式。

### 操作手法

#### 5.1 倒叙展开
```
原文（顺序）：
我们首先进行了数据收集，然后进行预处理，最后训练模型。

改写（倒叙）：
模型最终的表现令人满意，而这背后是一系列精心设计的流程——从数据采集的源头把控，到预处理阶段的多轮清洗和标注校验。
```

#### 5.2 论证方式切换
```
原文（纯正面论证）：
该方法能有效提升性能。实验中准确率提升了 5%。其他指标也有不同程度的改善。

改写（正反结合）：
如果不引入该方法，基线模型的准确率仅维持在 90% 左右。引入之后，这个数字跳升至 95%。不仅如此，召回率和 F1 值也随之水涨船高。
```

#### 5.3 夹叙夹议
```
原文：
实验分为三组。第一组使用方法A。第二组使用方法B。第三组使用方法C。结果表明方法B效果最好。

改写：
实验按方法差异分为三组。方法 A 作为基线方案中规中矩，方法 C 虽思路新颖但表现不够稳定。真正脱颖而出的是方法 B——它在准确率和稳定性两个维度上都明显占优。
```

## 模式六：学科适配

### 原理
不同学科有不同的写作传统和表达习惯，改写时应符合目标学科的风格。

### 适配要点

| 学科类型 | 风格特点 | 改写注意事项 |
|---------|---------|-------------|
| 理工科 | 精确、简洁、数据驱动 | 保持数据精确，少用修辞，可增加公式推导说明 |
| 人文社科 | 论证丰富、引用密集 | 可适当增加个人评述，引用更多文献，使用多样化论证 |
| 医学 | 严谨规范、术语密集 | 术语不可替换，着重改写连接性文字和讨论部分 |
| 法学 | 逻辑严密、条理清晰 | 可适当使用案例分析增加具体性 |
| 经管 | 数据与分析并重 | 增加具体案例数据，用图表说明代替文字描述 |

---

## 使用建议

1. **不要同时使用太多模式**：每个段落选用 2-3 种模式即可，过度改写反而不自然
2. **保持风格一致性**：全文风格应统一，不要某段口语化、某段极度正式
3. **先大后小**：先调整段落结构（模式五），再调句式（模式一二），最后改词汇
4. **朗读检验**：改写完成后默读一遍，如果读起来别扭就需要再调整
