# AI 检测工具特点分析

> 了解各检测工具的检测侧重点，有助于进行针对性优化。

---

## 一、国内主流检测工具

### 1. 知网 AIGC 检测

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于大语言模型判别器，分析文本生成概率分布 |
| **检测粒度** | 段落级别，标记每段的 AI 生成概率 |
| **检测优势** | 中文学术文本识别准确率高，数据库庞大 |
| **容易误判** | 模板化的文献综述、高度规范的方法论描述 |
| **针对性策略** | 重点改写摘要和结论部分，在方法论部分增加具体实验细节 |

### 2. 维普 AIGC 检测

| 项目 | 说明 |
|------|------|
| **检测原理** | 多模型融合检测，结合统计特征和语义分析 |
| **检测粒度** | 句子级别，逐句标注 AI 概率 |
| **检测优势** | 句子级别检测精度高 |
| **容易误判** | 简短的陈述句、数据描述性文字 |
| **针对性策略** | 逐句检查句式多样性，避免连续使用相同句型 |

### 3. 万方 AIGC 检测

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于困惑度和统计特征的综合检测 |
| **检测粒度** | 段落级别 |
| **检测优势** | 对困惑度异常低的文本特别敏感 |
| **容易误判** | 引用密集的段落、规范化的公式说明 |
| **针对性策略** | 提升文本困惑度，使用非常规但正确的词汇搭配 |

### 4. PaperPass AIGC 检测

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于语言模型的概率分析 |
| **检测粒度** | 段落级别 |
| **检测优势** | 中文互联网文本检测覆盖面广 |
| **容易误判** | 翻译体论文、教材式写作 |
| **针对性策略** | 增加原创性表述，减少翻译腔 |

## 二、国际主流检测工具

### 5. GPTZero

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于困惑度（Perplexity）和爆发度（Burstiness）双指标 |
| **检测粒度** | 句子级别，高亮标记可疑句子 |
| **核心指标** | Perplexity < 30 高度可疑，Burstiness < 0.5 高度可疑 |
| **针对性策略** | ① 提升困惑度：使用低频但正确的词汇 ② 增加爆发度：制造句子长度的剧烈波动 |

### 6. Turnitin AI Detection

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于大规模语言模型的细粒度检测 |
| **检测粒度** | 句子级别，给出整体 AI 生成百分比 |
| **检测阈值** | AI 生成占比 > 20% 标记为可疑 |
| **针对性策略** | 全面改写，重点关注连续 3 句以上的高可疑区域 |

### 7. Originality.ai

| 项目 | 说明 |
|------|------|
| **检测原理** | 多模型集成检测，号称检测率最高 |
| **检测粒度** | 句子级别 |
| **检测特点** | 对 GPT-4 生成的文本尤为敏感 |
| **针对性策略** | 需要深度改写，简单的同义词替换难以通过 |

### 8. ZeroGPT

| 项目 | 说明 |
|------|------|
| **检测原理** | 基于统计特征分析的快速检测 |
| **检测粒度** | 段落级别 |
| **检测特点** | 速度快但准确率相对较低 |
| **针对性策略** | 基本的句式变换和词汇替换即可通过 |

## 三、检测工具通用弱点

以下方面是大多数检测工具共同的盲区，可以利用：

1. **专业术语密集区域** — 检测工具难以判断是否为 AI 生成
2. **数据/公式描述** — 客观描述性文字不容易被识别
3. **直接引用** — 带引号的引用内容通常会被排除
4. **非标准表达** — 口语化、个性化的表达检测工具不熟悉
5. **跨语言文本** — 中英混杂的段落检测准确率下降

## 四、综合应对矩阵

| 如果目标检测工具是… | 优先使用的改写策略 |
|-------------------|-----------------|
| 知网 AIGC | 策略 2（词汇替换）+ 策略 3（人味注入）+ 策略 4（段落重组） |
| 维普 AIGC | 策略 1（句式变换）+ 策略 2（词汇替换） |
| 万方 AIGC | 策略 5（提升困惑度）+ 策略 1（句式变换） |
| GPTZero | 策略 5（提升困惑度）+ 模式二（长短句交替）— 增加爆发度 |
| Turnitin | 全面改写，策略 1-5 综合应用 |
| Originality.ai | 深度改写，策略 1-5 全部应用 + 策略 3（大量人味注入）|

## 五、注意事项

> [!WARNING]
> AI 检测技术更新迅速，本文档的信息基于截至 2025 年的检测工具表现。请定期测试和更新策略的有效性。

> [!NOTE]
> 没有任何改写策略能保证 100% 通过所有检测工具。目标是将 AI 检测率降低到合理范围内（通常 < 30% 被认为是安全的）。
