# AI 高频词/句式黑名单

> 以下词汇和句式是 AI 生成文本中出现频率极高的特征，在改写时应优先替换。

---

## 一、高频连接词/过渡词

### 🔴 必须替换（检测权重高）

| AI 高频表达 | 推荐替换方案 |
|------------|-------------|
| 首先…其次…最后… | 直接分段论述 / 用"一方面…另一方面" / 不用连接词直接切入 |
| 综上所述 | 回到最初的问题 / 从以上分析来看 / 总的来说（口语化）/ 直接给结论 |
| 值得注意的是 | 有意思的是 / 不容忽视的一点是 / 这里需要特别指出 / 直接陈述 |
| 需要指出的是 | 必须强调的一点 / 有一个关键细节 / 直接说明 |
| 总而言之 | 归结起来 / 简言之 / 一言概之 / 直接给结论不用过渡 |
| 与此同时 | 同期 / 在此期间 / 伴随着 / 另外 |
| 在此基础上 | 据此 / 以此为出发点 / 沿着这一思路 |
| 不仅…而且… | 除了…还… / …，同时也… / 拆成两个独立句子 |
| 然而/但是（句首） | 不过 / 可转念一想 / 事情并非如此简单 / 只是 |
| 因此/所以（句首） | 这就意味着 / 由此可见 / 这也就解释了为什么 |

### 🟡 建议替换（出现过多时需替换）

| AI 高频表达 | 推荐替换方案 |
|------------|-------------|
| 此外 | 另外 / 还有一点 / 除此之外 / 不用连接词 |
| 具体而言 | 说白了 / 拿具体例子来看 / 举例来说 |
| 鉴于此 | 考虑到这一点 / 既然如此 |
| 不难发现 | 显而易见 / 仔细看就会注意到 / 稍加分析就能看出 |
| 由此可见 | 这说明 / 可以看出 / 从中不难推断 |
| 换言之 | 也就是说（口语化）/ 说得直白些 |
| 事实上 | 实际上 / 就实际情况而言 / 其实 |

## 二、高频句式模板

### 🔴 必须改写

| AI 套路句式 | 问题分析 | 改写建议 |
|------------|---------|---------|
| "…具有重要的理论意义和实践价值" | 几乎所有 AI 生成的论文都有此句 | 具体说明是什么意义，举出实例 |
| "随着…的发展，…越来越…" | 万能开头模板 | 用具体事件/数据/引文开头 |
| "…的研究已经取得了显著进展" | 空洞无物 | 列举具体进展，引用具体文献 |
| "在…领域中发挥着重要作用" | 过于泛化 | 说明具体发挥了什么作用，怎么发挥的 |
| "…是一个复杂的系统工程" | AI 常用的万能总结 | 具体说明复杂在哪里 |
| "…面临着诸多挑战" | 空洞的过渡句 | 直接列出具体挑战 |
| "为…提供了新的思路和方法" | 万能结尾模板 | 具体说明提供了什么思路 |
| "本文旨在探讨…" | 机械化的目的陈述 | "本文关注的核心问题是…" / "这里要回答的问题是…" |

### 🟡 建议改写

| AI 套路句式 | 改写建议 |
|------------|---------|
| "研究表明…" | 指明是谁的研究、哪年做的，增加具体性 |
| "大量研究指出…" | 具体引用 2-3 篇关键文献 |
| "…引起了学界的广泛关注" | 列举具体的标志性事件或会议 |
| "…尚处于起步阶段" | 说明起步的具体表现，与成熟领域对比 |

## 三、高频词汇（单词级别）

### 文本中出现过多时会触发检测

| 类别 | AI 偏好用词 | 替换选择 |
|------|-----------|---------|
| **程度副词** | 显著、极大、充分、深入 | 有所、较为、一定程度上、在某种意义上 |
| **形容词** | 重要、关键、核心、有效 | 不可或缺的、举足轻重的、行之有效的、切实可行的 |
| **动词** | 探讨、分析、研究、阐述 | 考察、剖析、审视、梳理、论及 |
| **名词短语** | 提供参考、奠定基础、指明方向 | 提供借鉴、铺设路径、指出可能的方向 |
| **时间状语** | 近年来、目前、当前 | 过去十年间、截至 20XX 年、现阶段 |

## 四、英文论文 AI 特征词

> 如涉及英文论文改写，注意以下常见 AI 英文特征词

| AI 高频词 | 替换建议 |
|----------|---------|
| delve into | examine / explore / investigate |
| It is worth noting that | Notably / Interestingly |
| plays a crucial role | is central to / contributes significantly to |
| In conclusion | To summarize / Taken together |
| Furthermore | Additionally / Moreover / Also |
| a plethora of | many / various / numerous |
| utilize | use |
| facilitate | help / enable / support |
| comprehensive | thorough / detailed / extensive |
| underscore | highlight / emphasize |
| landscape | field / domain / area |
| tapestry | 避免使用此词，极其 AI 特征 |
| multifaceted | complex / diverse |
| paramount | essential / critical |

## 五、使用原则

1. **不要机械替换**：同一个黑名单词汇在不同语境下应选择不同的替换方案
2. **控制替换密度**：不是所有命中词都要替换，保留 10-15% 的常见词使文本更自然
3. **注意词汇搭配**：替换后需确保新词汇与上下文搭配合理
4. **学科敏感**：某些"AI 高频词"在特定学科中是标准术语，不应替换
5. **持续更新**：此黑名单应随着 AI 检测算法的更新而定期维护
